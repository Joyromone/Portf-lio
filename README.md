## Como inserir uma tela no HTML
Substitua:
  <div class="screen-slot"><div class="screen-slot__num">01</div></div>
Por:
  <div class="screen-slot"><img src="assets/tela-01.jpg" alt="Tela 01" /></div>

Nota: fotos pessoais (sobre mim, interesses) já estão embutidas em base64 no index.html.
